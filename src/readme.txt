What's about?
--------------
This script generates 2D s4 graphs for each constelation and frequency: GPS & GALILEO (default)
   -> INPUT: *.ismr files
   -> OUTPUT: *.pdf file
-------------
How it works?
    1) Place the INPUT files in the folder: ~/Input_data/Data_set/
    2) Run the script "plot_s4_2D.py"
    3) The OUTPUT file will be saved in the folder: ~/Output_data/
